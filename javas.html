Différentes Caractéristiques d'une variable

Elle est précédée d'un mot clé (const, var, let) qui prermet de le déclarer. Elle a un nom, Elle peut stocker différents types de données elle se termine par un point virgule.

Règles d'écriture pour les noms de variables

  • Une varible doit commencer par une lettre ou un underscore
  • Elle ne doit pas commencer par un chiffre, mais peut en contenir
  • Elle peut contenir des lettres, des chiffres et des undescrore
  • Elle ne doit pas contenir de caractère spéciaux
  • Le nom d'une variable est sensible à la casse

Les boucles

Servent à reproduire ou repéter du While loops A while loop performs a set of statements until a condition becomes false. These kinds of loops are best used when the number of iterations is not known before the first iteration begins. JavaScript has two kinds of while loops:
while evaluates its condition at the start of each pass through the loop.
do-while evaluates its condition at the end of each pass through the loop.

The while loop

A while loop starts by evaluating a single condition. If the condition is true, a set of statements is repeated until the condition becomes false.

The do-while Loop

The other variation of the while loop, known as the do-while loop, performs a single pass through the loop block first, before considering the loops condition. It then continues to repeat the loop until the condition is false.

For loops

For loops are used to run a set of tasks for a certain number of times. The for loop repeats a block of code as long as a certain condition is met. It is typically used to execute a block of code for a certain number of times.
for(initialization; condition; increment) { // Code to be executed }
  • Initialisation : is used to initialize the counter variables and evaluated once unconditionally before the first execution of the body of the loop.
  • Condition : : is evaluated at the beginning of each iteration. If it evaluates to true, the loop statements execute. If it evaluates to false, the execution of the loop ends.
  • Increment : : updates the loop counter with a new value each time the loop runs.

For-in loops

For-in loops are a special type of loop that iterates over the properties of an object, or the elements of an array.

Une fonction utilisateur

a function lets you define a block of code that performs a task. You give a function a name that identifies what it does, and this name is used to "call" the function to perform its task when needed.

Creating function

Every function has a function name, which describes the task that the function performs. To use a function, you “call” that function with its name and pass it input values known as arguments that match the types of the functions parameters. The code above is known as a function declaration. You define a function using the function keyword. After that comes the name of the function, followed by parentheses (). After the parentheses comes an opening brace, followed by the code you want to run in the function, followed by a closing brace. With your function defined, you can use it like so: printMessage();

Function Parameters

When you define a function, you can optionally define one or more named, typed values that the function takes as input, known as parameters. To make a function accept a parameter, you will need to add to function parentheses a parameter name. Below is a code example of a function that accepts two parameters. function outputName(name, lastName) { console.log(`${name} ${lastName}`); } You can define as many parameters as you like. However, for each parameter, you specify, a corresponding argument needs to be passed to the function when it is called otherwise, its value becomes undefined. Default values Parameters can have a default value. A default parameter value is used when a value was not provided during the function call.
